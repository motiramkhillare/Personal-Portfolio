# MotiRam Khillare — Portfolio

A premium, futuristic developer portfolio built with Next.js, React, TypeScript, Tailwind CSS, and Framer Motion.

## Getting Started

### Prerequisites

- **Node.js 18+** — Download from [nodejs.org](https://nodejs.org)

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```

## Editing Content

All content is centralized in **`lib/data.ts`**. Edit this single file to update:

- Personal info (name, email, social links)
- Skills and proficiency levels
- Projects (title, description, tags, links)
- Experience and education
- Contact information
- Terminal commands

### Adding Your Resume

Place your resume PDF at `public/resume.pdf`.

### Changing Profile Image

Replace `public/images/profile.png` with your own photo.

## Tech Stack

- **Next.js 15** — React framework
- **React 19** — UI library
- **TypeScript** — Type safety
- **Tailwind CSS 4** — Utility-first CSS
- **Framer Motion 12** — Animations
